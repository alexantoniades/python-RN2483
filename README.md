## RN2483 to Raspberry Pi GPIO connection
| Pin | RN2483 | to | Pin | Raspberry Pi 3B+ |
| --- | ------ | -- | --- | ---------------- |
| [7] | UART_RX | -> | [14] | UART_TX0 |
| [6] | UART_TX | -> | [15] | UART_RX0 |
